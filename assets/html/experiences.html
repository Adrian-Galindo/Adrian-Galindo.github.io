Stage Développeur Full-Stack - 2024


Description : Au sein de Perform Vision Training & Consulting, j’ai participé au développement d’un site web permettant aux utilisateurs de naviguer et de retrouver le formateur de leur choix. Le projet comprenait un système d’inscription et de connexion pour différents types d’utilisateurs (clients, formateurs, modérateurs, administrateurs), ainsi qu’un moteur de recherche et une messagerie intégrée.


Mes contributions :

  • 📌 Développement du système d’authentification : inscription, connexion et gestion des rôles (admin, formateur, client, modérateur).
  • 🔍 Implémentation d’un moteur de recherche permettant de filtrer et retrouver les formateurs selon plusieurs critères.
  • 💬 Développement d’un système de messagerie client/formateur pour faciliter la communication.
  • 🖊️ Mise en place de la signature électronique pour la validation des documents.
  • 📄 Génération automatique de PDF à partir des informations collectées.
  • 📊 Optimisation et amélioration de la base de données pour assurer de meilleures performances.

Projet universitaire - Sélecteur multimodal - 2025


Description : Dans ce projet, nous avons développé Peinturia, une application web permettant aux utilisateurs de découvrir des œuvres d'art, d'apprendre et d'acheter des matériaux de qualité, tout en ayant une conscience écologique. Le projet a été centré autour d'une recherche multimodale avancée et d'une navigation intuitive, offrant une expérience utilisateur optimale.


Mes contributions :

  • 🔍 Développement de la fonctionnalité pipette permettant d’extraire une couleur à partir d’une œuvre d’art.
  • 📊 Scraping de données pour récupérer les noms de peintures et les œuvres d’art.
  • 🖥️ Développement d’une API (première version) pour structurer et exploiter les données collectées.
  • 🎨 Automatisation du traitement des images des produits (peintures) avec Python.
  • 🗄️ Génération de scripts SQL pour l’implémentation des données :
    • Automatisation de la création de fichiers CSV avec Python.
    • Conversion des fichiers CSV en SQL pour la gestion de la base de données.
